title: Official API
description: The Official surface — the controller's own integration/v1 OpenAPI, reached via c.Official() and addressed by site UUID.
---

The **Official API** is the controller's own [UniFi Network API](https://developer.ui.com) (`integration/v1`),
reached from any `unifi.Client` via [`c.Official()`](/docs/reference/official-api/official). It is a separate,
fluent surface — one accessor per resource group — that addresses sites by `uuid.UUID` rather than the legacy
site-name string.

It requires **API-key auth** and a newer controller: the Official surface needs **10.1.78** or later (API keys
themselves are accepted from 9.0.114). The first call probes `GET /v1/info` once and then fails fast if the
controller is too old, unreachable, or disabled. See [Choosing a surface](/docs/guides/choosing-a-surface) for
when to prefer it over the Internal API.
